# CLAUDE.md

This is a DayZ mod development repository. The mod (`mod/`) is written in Enforce Script (DayZ's scripting language).

## Skills

- **`/dayz-logs`** — Read, search, list, or follow DayZ script logs. Use this whenever the user asks to check logs.

## References

A modding wiki is available at `./references/modding/`. It covers Enforce Script, mod structure, GUI system, and more. Consult it when working with DayZ-specific APIs or patterns.

## Current mod being worked on

**SSCollector**/**dayz-ss-collector** is a mod that will aid in collecting many labeled screenshots from the game for machine learning purposes. The plan is for it to cover teleporting, metadata saving on request and setting weather and time.
